# Agent Operating Contract

This project uses the Paperclip-UAW workflow. Read and follow `CLAUDE.md` in this directory for the full operating contract, including:

- Authority order
- Session start/end protocol
- Decision classification and governance rules
- Phase classification and verification depth
- Multi-agent pipeline rules

All rules in CLAUDE.md apply to all agents regardless of tool (Claude Code, Codex, Gemini, etc.).
